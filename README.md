## Descripción del Proyecto

Este repositorio contiene el flujo de trabajo analítico (**End-to-End Data Pipeline**) implementado en **Databricks** para la red de clínicas **Panama Metropolitan Medical (PMM)**.

A partir de una base transaccional simulada de **10,000 citas médicas** correspondientes al año 2025, se construyó un pipeline completo que abarca desde la ingesta de datos en bruto hasta la publicación de un **Dashboard Interactivo** para la toma de decisiones gerenciales y la implementación de un modelo predictivo de precios mediante **Machine Learning (XGBoost)**.
